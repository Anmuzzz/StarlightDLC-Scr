## 🚀 Функционал (Features)

В клиенте реализовано множество механик — от классических хаков до продвинутых систем кастомизации игрового процесса:

* **Combat (Бой):**
  * **KillAura** — имеет раздельные режимы работы: под старое ПВП (версии 1.8) и под новые версии Minecraft с учётом задержки ударов.
  * **TriggerBot** — автоматический удар при наведении прицела на цель.
* **Visuals (Визуалы):**
  * **Кастомные модельки** — поддержка изменения и кастомизации внешнего вида моделей игроков/сущностей.
  * Трейсеры, ESP (подсветка), Fullbright (вечное зрение в темноте) и другие элементы рендеринга.
* **World / Player:**
  * **Автосвап (AutoSwap)** — автоматическая смена предметов или оружия в руке для максимальной эффективности.
  * И множество других вспомогательных модулей.
* **Scripting (В разработке):**
  * **LUA Скриптинг** — заложена база для создания кастомных плагинов и скриптов (на данный момент *требует фикса, система не активна*). Отличная точка для тех, кто хочет поковырять код и дописать API.

## 🛠️ Требования и Установка

### Для игроков (Готовый `.jar`)
1. Установите **Fabric Loader** для Minecraft 1.21.4.
2. Скачайте **Fabric API** для 1.21.4 и закиньте его в папку `.minecraft/mods/`.
3. Перейдите в раздел **Releases** на этой странице (справа), скачайте скомпилированный `.jar` файл чита и также положите его в папку `mods`.
4. Запустите игру и откройте меню чита на клавишу **[RSHIFT]** (Правый Shift) или **[Y]**.

### Для разработчиков (Сборка из исходников)
Проект использует стандартную систему сборки Gradle (Fabric Loom).

1. Клонируйте этот репозиторий:
   ```bash
   git clone [https://github.com/Anmuzzz/StarlightDLC-Scr.git](https://github.com/Anmuzzz/StarlightDLC-Scr.git)
   cd StarlightDLC-Scr
Скомпилируйте модификацию:

Windows: gradlew.bat build

Linux/macOS: ./gradlew build

Готовый скомпилированный .jar файл появится в директории build/libs/.
